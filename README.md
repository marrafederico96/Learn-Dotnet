# 🎉 Friend Stuff - WIP

**Friend Stuff** è una piattaforma pensata per gruppi di amici, con l'obiettivo di centralizzare l'organizzazione e la gestione di eventi privati in modo semplice e collaborativo.

---

## 🚧 Stato del Progetto

🔧 **Work in progress** – il progetto è in fase di sviluppo attivo.

- 👥 Creazione di gruppi
